**2. Chosen Application**
The application our group recreated is **GCash**.

GCash is a mobile wallet application in the Philippines that allows users to perform financial transactions such as sending money, paying bills, buying load, and scanning QR codes for payments.

For this project, our goal was to recreate the **user interface of the GCash home screen using React Native with Expo**. The focus of the project was to replicate the layout, icons, and design of the application, rather than implementing real financial features.

**4. Problems Encountered**
During the development of the project, our group encountered several issues:

* Dependency conflicts while installing npm packages.
* Errors related to Expo SDK compatibility and package versions.
* Missing Babel configuration that caused the project to fail when running.
* Metro bundler cache issues that required clearing the cache.
* Difficulty matching the layout and spacing of the original GCash interface.

These issues were resolved by correcting dependency versions, reinstalling packages, and properly configuring the project.
